Extended Kalman Filter (EKF) class for state estimation.

This class implements the Extended Kalman Filter (EKF) algorithm for state estimation in a non-linear system. It takes simulation data, configuration, atmosphere model, time step, process noise covariance, measurement noise covariance, initial error covariance, and measurement matrix as input and returns the estimated trajectory and measurements.
